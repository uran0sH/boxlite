# BoxLite logo

Source brand assets, each in SVG (scalable) and transparent PNG.

| File | Use |
|------|-----|
| `boxlite-icon.*` | icon only — app marks, favicons, avatars |
| `boxlite-word.*` | wordmark only |
| `boxlite-lockup.*` | horizontal lockup (icon + wordmark) — the default logo |
| `boxlite-lockup-stacked.*` | vertical lockup — square / centered placements |

**Colorway:** black outlines + blue, for light backgrounds. On dark backgrounds,
recolor the black to white (the blue stays).

The full-width README banners live in `.github/assets/boxlite-banner-{light,dark}.png`.
